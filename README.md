# 项目功能
为缩短链接提供一个美观、易用的Web前端界面。

# 前端技术栈
Vue3 + Typescript + Axios + Sass

# 项目规范
使用了`husky`、`pre-commit`和`lint-staged`、`cz`作为提交检查器。提交规范如下：
- feat - 新功能 feature
- fix - 修复 bug
- docs - 文档注释
- style - 代码格式(不影响代码运行的变动)
- refactor - 重构、优化(既不增加新功能，也不是修- 复bug)
- perf - 性能优化
- test - 增加测试
- chore - 构建过程或辅助工具的变动
- revert - 回退
- build - 打包

# 代码质量检查工具
- `prettier`
- `eslint`

# 项目编译 & 开发依赖安装
- `pnpm i`
- `pnpm run prepare`
